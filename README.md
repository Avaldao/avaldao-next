# AvalDAO App (Next Generation)

This repository contains the new version of the AvalDAO application, built with **Next.js** and the latest **WalletConnect** integration.

It replaces and consolidates the legacy projects:

- `avaldao-dapp`
- `avaldao-feathers`

This new architecture unifies the frontend and backend layers into a modern Web3 stack, enabling users to securely connect their wallets and interact with the AvalDAO platform.
